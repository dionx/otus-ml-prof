# About Dataset

Набор данных IMDB, содержащий 50 000 отзывов о фильмах, предназначен для обработки естественного языка или анализа текста.
Это набор данных для бинарной классификации настроений, содержащий значительно больше данных, чем предыдущие эталонные наборы данных. Мы предоставляем набор из 25 000 высокополярных отзывов о фильмах для обучения и 25 000 для тестирования. Таким образом, можно прогнозировать количество положительных и отрицательных отзывов, используя алгоритмы классификации или глубокого обучения.
Для получения дополнительной информации о наборе данных, пожалуйста, перейдите по следующей ссылке:
http://ai.stanford.edu/~amaas/data/sentiment/
